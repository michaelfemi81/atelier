# Atelier Lane

A modern Next.js starter for a clothing shopping app with two experiences:

- Customer storefront at `/`
- Owner admin dashboard at `/admin`

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Current screens

- Storefront homepage with hero, categories, new arrivals, and service features
- Admin dashboard with store stats, product upload form, product list, and order queue preview

## Suggested next steps

- Connect Supabase for auth, product data, and image storage
- Add real product creation from the admin form
- Add product detail, cart, checkout, and order management routes
- Add payment integration with Stripe, Paystack, or Flutterwave
